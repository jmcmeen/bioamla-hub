# Tools

Here we list some tools we use in our projects for bioacoustics, machine learning, software development, and data science.

- [Raven](https://www.birds.cornell.edu/ccb/raven-pro/) - Software program for the acquisition, visualization measurement, and analysis of sounds
- [Spotlight](https://renumics.com/open-source/spotlight) - Open source data curation tooling for unstructured data
- [VS Code](https://code.visualstudio.com/) - Lightweight code editor with extensions for language support
